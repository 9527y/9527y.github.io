各个系统查看端口情况的方法

June 2, 2020

查看本机占用情况

windows

windows常用命令netstat，与find组合使用

netsat -an | find "8080"

mac

lsof -i 8090

linux

netstat -anptl | grep 8080

查看远程端口是否开启

nc

nc -v -n 127.0.0.1 22

wget

 wget 192.168.1.1:6379

curl

curl 192.168.1.1:6379
