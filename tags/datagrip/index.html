Datagrip导入导出数据库设置

导出数据库设置 选中想要导出数据设置 呼出热键命令 cmd+shift+a 输入 Database Tools 选中 Copy Settings 然后粘贴到文本中就可以保存配置 导入数据库配置 复制到导出的配置内容，可以分段拷贝配置 呼出热键命令 cmd+shift+a 输入 Import from Clipboard 注意 剪切板导入的数据库配置，可以另存为文件 导出的配置没有密码，需要自行重新配置密码 https://stackoverflow.com/questions/37931260/how-to-export-import-a-data-source-from-datagrip
